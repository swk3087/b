# PlanRiseAI 소개 영상용 핵심 기능

## 한 줄 소개
- PlanRiseAI는 계획을 세워주는 것을 넘어, 매일 실행하게 만드는 AI 학습 코치입니다.

## 주요 기능
1. AI 자동 플랜 생성
- 남은 페이지, 마감일, 요일별 가능 시간을 바탕으로 날짜별 학습량을 자동 배정합니다.

2. 실행 중심 홈 화면
- D-Day, 오늘 할 일, 밀린 항목, 응원 문구를 한 화면에서 바로 확인합니다.

3. 미완료 자동 재분배
- "오늘 할 일 다음으로 미루기"로 밀린 분량을 내일부터 자동 재배치합니다.

4. 캘린더 직접 편집
- 캘린더에서 일정의 제목/날짜/페이지/시간/상태를 수정하고, 개별 삭제 및 전체 삭제가 가능합니다.

5. 설정 기반 개인화
- 휴식 패턴, 버퍼 비율, 오프데이, 요일별 가능 시간을 설정해 현실적인 계획으로 조정합니다.

6. 플랜 미리보기 후 저장
- 생성 즉시 저장하지 않고, 미리보기 확인 후 사용자 승인 시에만 캘린더에 반영합니다.

7. 푸시 알림 지원(PWA)
- 앱 설치, 푸시 구독, 서버 푸시 테스트로 실행 리마인드를 제공합니다.

## 요금제 포인트
- 무료: AI 플랜 2회/월, 재분배 2회/월
- 월간: AI 플랜 6회/월, 재분배 10회/월
- 연간: AI 플랜/재분배 무제한
