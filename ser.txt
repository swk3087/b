PlanRiseAI 서비스 개요서


- PlanRiseAI = ‘계획을 자동으로 짜고 + 매일 하게 만드는’ AI 플래너/코치
- “문제풀이 패턴 분석” 같은 고난도 개인화는 초기엔 포기(데이터/전문가/저작권 이슈 큼)
- 대신 실행을 강제하는 기능(루틴, 알림, 시간 압박, 버퍼 포함 일정)에 집중
- AI는 창의적인 상담보다 ‘일정 자동 분배 + 재조정’에만 사용


- 중·고등학생 내신/문제집
- 성인 자격증/시험 준비(확장 가능)


- 오늘 목표를 마감 형태로 제시
- 남은 시간 대비 남은 분량을 보여주고 미션처럼 완료 체크

- “사용자 시간을 다 채우지 않기”를 규칙화
- 예: 하루 가능 3시간이면 2시간 20분만 배정(버퍼/휴식 포함)

- 미수행 시 ‘밀린거 자동 정리 + 다음날 재분배’
- 홈에 ‘오늘 할 일/밀린거/D-day’를 항상 노출

- 매주 명언/응원 문구를 UI 일부로 업데이트
- “오늘 아무 것도 안 했으면 우울” → 부담 덜한 응원 + 다시 시작 버튼

B. 데이터 전략(‘문제풀이 패턴 분석’ 대신 현실적 접근)
-  문제집 PDF/정답/난이도 데이터를 학습시키지 않음(비용/저작권/확장 어려움)
- 대신 사용자 행동 데이터로 개인화:
  * 계획 대비 실제 완료율
  * ‘이 문제잡 어려움/쉬움’ 1~5 평가
  * 실제 소요시간(예상 vs 실제)
- 위 데이터만으로도 다음 계획 분배 정확도 개선 가능

5) 화면/플로우(반영 버전)

1) 초기질문(기존 + 실행 중심 질문 추가)
- 기존: 학교, 성적구간, 요일별 가능 시간, 오프데이(서버 저장, 설정에서 변경)
- 추가:
  * 공부 스타일: 빡세게 몰아치기 / 꾸준히
  * 독촉 강도: 부드럽게 / 보통 / 강하게
  * 휴식 선호: 50/10, 25/5 등(옵션)

2) AI플래너(플랜 생성)
- 문제집 끝내기 / 내신 대비하기(시험 3일 전까지 완주)
- 추가:
  * 실전 압박 모드 ON/OFF
  * 버퍼 비율(예: 15~25%) 기본값 제공

3) 홈(핵심 고정)
- D-Day
- 오늘 할 일(타이머/압박형 표시 가능)
- 밀린거(자동 재분배 버튼)
- (선택) 오늘의 응원/명언 카드

6) AI(OpenAI) 사용 범위(최소화)
- AI가 하는 일:
  * 일정 분배(페이지/분)
  * 오프데이 제외
  * 버퍼 포함(시간 꽉 채우지 않음)
  * 밀린 분량 발생 시 재계산(리플래닝)
  * 이전 계획 vs 변경 계획 생성(비교용)

설정페이지, 내계정 페이지도 만들기.

오늘 할일 다음으로 미루기 버튼도.(재분배)



- AI가 안 하는 일(초기):
  * 문제집 내용 이해/난이도 판정/정답 데이터 기반 추천(PDF/저작권/비용 이슈)

7) MVP 우선순위(빠르게 출시)
- MVP = 계획 생성 + 매일 실행시키기
  1) 플랜 생성(문제집/내신)
  2) 홈(오늘할일/밀린거/D-day)
  3) 알림/독촉
  4) 휴식/버퍼 로직
  5) 미완료 시 자동 재분배
- 이후:
  * 사용자 난이도 평가(별점) → 계획 품질 개선
